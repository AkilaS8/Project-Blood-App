Android Studio

For develop this mobile application , used Android Studio with JAVA.

Firebase

Firstore Database service on Firebase used as database.

Adobe XD

Adobe XD used for design front-end of this mobile application.

Features



Responsive Design

This application design responsive to any kind of android device screen size.

Support to the lowest and the latest versions

This application can install the lowest Android version API level 19 (Andriod 4.4 Kitkat version) to the latest API levels.

Provide Emergency Services

This app provide emergency service numbers, ambulance service numbers, blood bank details and so on.



Alternative Results

If user seeking O+ blood, then user can also use O- blood as alternative. That results also show when user search.

Nested Spinners

This app has nested spinners to enter your location. first spinner provide Province, second District and third provide city.

Contact Donor Easily

By clicking donor app pop-up donor details with contact nummber and diresction links. By clicking contact number easily can get call and by clicking diresction link it shows diresction from your loaction.

Blood Donatye App

This mobile application developed according to the client's requirements. By using this app users can find donors or the user can request blood. Also, this app provides so many services like ambulance services numbers and so on.

For adding location this app used nested spinners. Firest spinner provides to select a province. Then the second spinner fills districts according to the user selection of the first spinner. Then the third spinner fills cities according to the second spinner selection. Also, this app shows alternative results for searchers. This app has an awesome user interface with beautiful radio buttions.
